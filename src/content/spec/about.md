
本网站使用 **Astro** 框架构建，采用 [Mizuki](https://github.com/captain-ht/Mizuki) 主题。

::github{repo="captain-ht/Mizuki"}

## 🌟 主题特性

### 🎨 设计与用户体验
- **现代优雅的设计** - 简洁、极简的界面，配有精美的排版
- **完全响应式** - 针对从移动设备到桌面的所有设备进行优化
- **深色/浅色模式** - 自动主题切换，过渡流畅
- **精美的排版** - 使用 JetBrains Mono 字体增强可读性
- **流畅的动画** - 流畅的页面过渡和交互元素

### 🔍 内容与搜索
- **高级搜索** - 由 [Pagefind](https://pagefind.app/) 提供支持，快速、准确的结果
- **增强的 Markdown** - 扩展语法，支持代码高亮和数学公式
- **交互式目录** - 长文章的自动滚动导航
- **RSS 订阅生成** - 自动生成订阅源，保持更新
- **阅读时间估算** - 了解文章阅读所需时间
- **文章分类** - 使用标签和分类组织内容



### 📱 特色页面
- **追番记录页面** - 跟踪你的追番进度和评分
- **友链页面** - 用精美的卡片展示友链网站
- **日记/动态页面** - 像社交媒体一样分享生活瞬间
- **归档页面** - 所有文章的有序时间线视图
- **关于页面** - 可自定义的个人介绍（本页面！）

### 🛠 技术特性
- **增强的代码块** - 由 [Expressive Code](https://expressive-code.com/) 提供支持
- **数学公式支持** - 使用 KaTeX 渲染 LaTeX 数学表达式
- **图片优化** - PhotoSwipe 图库，支持懒加载
- **SEO 优化** - 内置站点地图和元标签，提升搜索可见性
- **性能优化** - 快速加载，支持缓存和优化
- **评论系统就绪** - 支持 Twikoo 评论集成

### 🎯 高级 Markdown 特性
- **提示框与标注** - 使用 `> [!NOTE]`、`> [!TIP]`、`> [!WARNING]` 创建精美的信息框
- **数学公式** - 使用 `$inline$` 和 `$$block$$` 语法编写 LaTeX 数学公式
- **GitHub 卡片** - 使用 `::github{repo="user/repo"}` 嵌入仓库卡片
- **语法高亮** - 带行号的高级代码高亮
- **复制代码按钮** - 便捷的代码复制功能
